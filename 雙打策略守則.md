# 雙打比賽策略守則

## 基本資訊

- 賽制：男雙（混雙組合）
- 讓分：開局 2:0 領先（我們有優勢）
- 角色：我（進攻主力）、學妹（控制主力）

---

## 角色分工

| 角色 | 我 | 學妹 |
|------|----|------|
| 主要任務 | 發球得分、進攻收尾 | 接發球控制、銜接過渡 |
| 搶攻責任 | 第三板、第五板 | 銜接過渡，不主動冒進 |
| 配球方向 | 壓反手、找空檔 | 回球壓底線或回短 |

---

## 發球守則（每局順序）

| 球序 | 發球種類 | 目的 |
|------|----------|------|
| 第1球 | 短轉不轉 | 試探對手接發習慣 |
| 第2球 | 短轉不轉 | 確認對手弱點 |
| 第3球 | 側旋 or 下旋 | 換旋轉讓對手不適應 |
| 第4球 | 奔球 | 突擊，打亂節奏 |
| 第5球以後 | 針對弱點重複 | 持續施壓 |

> 每局第一球永遠試探，不要上來就奔球賭博

### 我的發球戰術鏈

**短轉不轉**
- 對手搓短回來 → 擰拉正手大角
- 對手搓長回來 → 側身暴拉
- 對手挑起 → 正手搶拉

**左側旋**
- 對手回球往學妹正手位 → 學妹進攻或我繞過去打
- 對手回直線到我反手 → 我反手或側身接

**砍式下旋**
- 對手拉球下網 → 直接得分
- 對手搓回長球 → 我側身暴拉
- 對手拉起來 → 看落點反拉或側身

**奔球（關鍵分使用）**
- 對手接不到 → 直接得分
- 對手勉強回長球 → 我直接進攻

### 學妹的發球戰術鏈

**反手短下旋**
- 對手搓回中路 → 我上前拉球進攻
- 對手搓回反手 → 學妹銜接控制
- 對手冒高挑起 → 我直接正手扣殺

**勾手側上（對陌生對手效果最好）**
- 對手下網或回球偏高 → 我直接扣殺
- 對手回球偏弱 → 學妹或我進攻收尾

---

## 搶攻守則

| 來球情況 | 處理方式 |
|----------|----------|
| 台內短球 | 擰拉，打反手大角或直線 |
| 台中長球（下旋） | 側身暴拉 |
| 台中長球（上旋） | 正手拉，壓反手位 |
| 來球偏高 | 直接扣殺，不手軟 |

### 搶攻三板斧

**第一斧：擰拉搶節奏**
- 時機：台內短球
- 打法：擰拉壓反手大角或直線
- 目的：讓對手第一板就被動

**第二斧：側身暴拉**
- 時機：來球到中路或偏正手
- 打法：側身正手暴拉斜線
- 目的：最大力量和角度，直接得分

**第三斧：連續壓角**
- 時機：對手回球到台中
- 打法：第一板壓反手，第二板轉正手空檔
- 目的：拉動對手後找空檔收尾

### 擰拉使用時機

優先擰拉：
- 對手接發球回短球
- 學妹搓回的短球被對手回到我這邊
- 開局前幾分，需要氣勢球
- 領先 3 分以上，可以積極出手

暫緩擰拉：
- 來球旋轉不確定
- 站位沒到位
- 9 分後關鍵分
- 平分或落後時（先求進台）

---

## 每一分的決策流程

1. 發球製造機會
2. 第三板：擰拉 or 拉球搶節奏
3. 第五板：壓角或收尾
4. 對手回弱球：立刻扣殺，不猶豫

### 配合範例

**我發球輪**

```
我發短轉不轉
→ 對手搓回短球
→ 我擰拉壓反手大角
→ 對手被動回中路
→ 學妹快帶過渡
→ 我側身暴拉正手空檔收尾
```

**學妹發球輪**

```
學妹發短下旋
→ 對手搓長到我正手
→ 我正手拉斜線壓反手
→ 對手回直線到學妹
→ 學妹反手控短或快撥
→ 對手回球偏弱
→ 我正手收尾
```

## 比分管理守則

| 比分狀態 | 策略 |
|----------|------|
| 2:0 開局 | 不急，讓對手先慌 |
| 領先 3 分以上 | 繼續積極，拉開差距 |
| 領先 1～2 分 | 穩定發球，不冒險擰拉 |
| 平分 | 重置心態，一球一球打 |
| 落後 | 換發球種類，打亂對手節奏 |
| 9 分後 | 回歸最穩發球，不試新招 |
| 10:10 | 忘記讓分，當 0:0 打 |

---

## 針對不同對手

| 對手類型 | 應對方式 |
|----------|----------|
| 強力正手進攻型 | 學妹發短下旋壓反手，不給機會球 |
| 削球手 | 多用奔球＋擰拉，不打搓球拉鋸 |
| 雙反手型 | 發側旋讓球往正手跑，找正手空檔 |
| 穩健老練型 | 加快節奏，奔球破節奏，不打慢長拍 |

---

## 禁止事項

- 拉球只求進台，沒有落點和速度
- 輪轉還沒到位就搶打
- 連續用同一種發球超過 3 次
- 9 分後還在試新招
- 中路球猶豫不喊
- 擰拉位置站不到還勉強出手

---


## 核心原則

> **發球試探 → 擰拉搶節奏 → 壓角 → 收尾**
>
> 學妹用落點把對手打亂，我用擰拉和側身暴拉收尾
>
> 我們不是在追分，是在守分
